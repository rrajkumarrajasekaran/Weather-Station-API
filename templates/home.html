Weather Station API

For Temperature of One station, Recorded One date:

API URL Format: http://127.0.0.1:6234/api/v1/`station`/`date`

Example: http://127.0.0.1:6234/api/v1/11/1996-04-10

For Temperature of One station, All Dates dates:

API URL Format: http://127.0.0.1:6234/api/v1/`station`

Example: http://127.0.0.1:6234/api/v1/11

For Temperature of One station, For a given year:

API URL Format: http://127.0.0.1:6234/api/v1/yearly/`station`/`year`

Example: http://127.0.0.1:6234/api/v1//yearly/11/1996
